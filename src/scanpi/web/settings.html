Edit config

Runtime values above are read-only at the moment. To change them, edit the tool config and restart the service: 
# edit config (example — advanced users):
sudo journalctl -u scanpi.service -n 20
sudo systemctl edit scanpi.service        # inject Environment= overrides
sudo systemctl restart scanpi.service
A future release will let you change gain / squelch / retention budget from this page directly. For now the values below update live from /api/tools so you can confirm what the daemon loaded.
